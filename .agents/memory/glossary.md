# Memory: Domain Glossary

## Core Domain & Technical Terms

- **Sari-Sari Store**: A neighborhood convenience store in the Philippines, typically family-owned, offering retail goods, snacks, household essentials, and mobile load.
- **Suki**: A loyal or regular customer of a sari-sari store.
- **Utang Ledger**: The informal credit record maintained by store owners to track customer debts, purchases, and repayments.
- **Offline-First**: A software architecture designed to function fully without active internet connectivity, synchronizing data asynchronously when a network becomes available.
- **Active Persimmon**: The primary brand accent color (`#E85A1F`) representing a physical stamp mark of confirmation on paper receipts.
- **Cream Paper**: The neutral canvas background color (`#FBF7EE`) representing organic ledger paper.
- **Dark Ink**: The high-contrast typographic color (`#0E0C0A`) used for maximum text legibility on mobile displays.
- **Taglish**: A natural blend of Tagalog and English commonly spoken in the Philippines, used across website copywriting to maintain an accessible and friendly tone.
- **Vercel Serverless Function**: API handlers residing in the `api/` directory that execute server-side Node.js logic on Vercel infrastructure.
