# Sari Master Prompts for Character Design

This document contains the finalized image generation prompts for Sari, the mascot of SariSari. These prompts use a hybrid approach combining a **Core DNA** (for character consistency) and a **Technical Wrapper** (for game-asset quality), with specific **State Modifiers** to ensure correct emotional expression.

## 🛠 Prompt Construction Logic

Each prompt is assembled as follows:
`[Core DNA]` + `[State Modifier]` + `[Technical Wrapper]`

**Core DNA:** A high-quality 2D game sprite of a character named Sari. Sari is a minimalist anthropomorphic cylindrical glass jar (garapon) with a large rounded bottom. She has a flat-top lid acting as a hat in terracotta orange (#C2410C). She has large, minimalist dot-eyes. Her limbs are thin, rounded vector lines in a warm charcoal gray. Clean vector art, flat colors, consistent line weights, centered composition.

**Technical Wrapper:** Isolated on a plain white background. Flat 2D vector style. Full body shot. Game asset, sprite sheet quality. No gradients, no 3D rendering, no shadows, no atmospheric lighting. High contrast, sharp edges.
